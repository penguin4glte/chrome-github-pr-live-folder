GitHub PR Live Folder 設定

GitHub アカウント

Private リポジトリを含む場合は repo スコープ（fine-grained なら Pull requests: Read）が必要です。 組織のプライベートリポジトリには、トークンのSSO認可（組織がSAML SSOを使う場合）が必要です。 トークンはこの端末の拡張機能ストレージにのみ保存されます。

同期設定

GitHub の検索構文が使えます。例: is:pr is:open review-requested:@me / is:pr is:open org:yappli author:@me / is:pr is:open repo:owner/repo
